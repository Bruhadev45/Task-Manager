# Screenshots

This directory should contain screenshots of the working application.

## Required Screenshots

1. **task-list.png** - Screenshot of the Task List Page showing all tasks
2. **create-task.png** - Screenshot of the Create/Edit Task Page with the form

## How to Take Screenshots

1. Start the application:
   ```bash
   ./start.sh
   ```

2. Open http://localhost:3000 in your browser

3. Take screenshots:
   - **Task List Page**: Take a screenshot of the main page showing tasks
   - **Create Task Page**: Click "Create New Task" and take a screenshot of the form

4. Save the screenshots in this directory with the names:
   - `task-list.png`
   - `create-task.png`

## Note

Screenshots should demonstrate:
- Clean, working UI
- All required fields visible
- Tasks displayed with their status, priority, and due dates
- Form with all required fields (title, description, status, priority, due_date)
